# 07 · Platform Adaptation Strategy

**One core video → three platforms.** Shoot/edit once; adapt only the seven surface layers below. Do **not** make three different videos per topic.

---

## What changes per platform (the only seven levers)

| Lever | Instagram Reels | TikTok | YouTube Shorts |
|---|---|---|---|
| **Hook style** | Visual + soft text hook; aesthetic first frame | Verbal hook in first second; punchier | Keyword-spoken hook that matches the title |
| **On-screen title** | Minimal, elegant (lean on the cream brand) | Bold, high-contrast, casual | Present, but the metadata title carries discovery |
| **Caption length** | 1–3 lines + the prompt | 2–4 conversational lines, search phrase first | Short; title does the work |
| **CTA** | "link in bio" | "link in bio" (say it + on-screen) | "check my profile / link" (bio links differ) |
| **On-screen text** | Refined, sparse | Native-style captions, more of them | Clear captions; readable at a glance |
| **Search keywords** | Hashtags (niche+mid) | Search phrases in caption + spoken + on-screen | Title + description keywords (highest weight) |
| **Description** | Short | Medium, conversational | Real 2–4 line keyword description |

Everything else — the footage, the VO, the core lesson, the end card — stays identical.

---

## Platform personalities (keep in mind)

- **Instagram Reels** — the brand's showcase. Aesthetics matter most; the cream/serif look is a competitive edge here. Saves and shares drive reach. Bio link is single/native — keep it pointed at `guide.delightdigital.online`.
- **TikTok** — fastest discovery, most forgiving of "rough." Front-load a spoken hook; TikTok search rewards keywords in caption + spoken words + on-screen text. Slightly more casual voice is fine.
- **YouTube Shorts** — the evergreen library. A searchable title + real description gives long-tail discovery months later. Best home for "how to" and "for beginners" framing. Because Shorts bio links are less prominent, drive to profile → link.

---

## Posting rhythm (solo creator)
- **5 posts/week (Mon–Fri)** across all three platforms simultaneously = one core video/day, three uploads.
- Batch production by **format** (see `04`), not by day.
- ~6 calendar weeks to run all 30 (or a daily 30-day sprint for a launch push).
- Reserve weekends for filming/editing the next batch and replying to comments (comment replies → future "comment-response" videos).

---

## Cross-posting hygiene
- Re-export per platform where possible (avoid visible competitor watermarks — can suppress reach).
- Keep the safe zones clear: TikTok/Reels UI covers the bottom ~15% and right side — keep captions and the end-card CTA out of those zones.
- Same brand end card everywhere; only the spoken/último CTA wording shifts per the table.
