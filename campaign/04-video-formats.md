# 04 · Repeatable Faceless Video Formats

Seven templates one person can produce efficiently. All are **faceless** (no on-camera person, no filming another person, no copyrighted clips, no showing your face). Reuse the same brand kit every time so the feed reads as one calm system.

**Brand kit (from the products):** cream/ivory `#F2EEE6` and espresso `#2B2320` backgrounds · terracotta/dusty-rose accent · editorial **serif** for big headlines (italic accent on the last word) · clean **sans** for body · **monospace** for small labels/prompt boxes · generous whitespace · calm, quiet pacing. Captions always on (accessibility + sound-off viewing).

---

## Format 1 — Interactive-site screen recording
- **Visual structure:** clean screen capture of `ai.` or `workbook.` → cursor fills a field / clicks copy / checks a day → show the result. Add a title card in and an end card out.
- **Duration:** 25–40s. **Difficulty:** low.
- **Voiceover:** calm VO or captions-only over soft music.
- **Captions:** monospace step labels ("Step 1 · pick a task").
- **B-roll / capture:** record at phone aspect (1080×1920) or crop a desktop capture; hide any personal data.
- **Best use case:** showing the product is real. **Best for:** clicks / sales.
- **Template move:** always show *one* feature per video (the builder, OR the prompt library, OR the workbook fill-in) — never a full tour except Day 29/30.

## Format 2 — Before/after prompt (Prompt Doctor)
- **Visual structure:** split screen — left "DON'T" (vague one-liner) / right "TRY" (the 5-ingredient version) → paste the good one → reveal a better answer.
- **Duration:** 20–30s. **Difficulty:** low.
- **Voiceover:** read both prompts; land the one-line lesson.
- **Captions:** the two prompts as monospace cards; highlight the added ingredients in terracotta.
- **B-roll:** screen capture of the chat, OR static styled cards.
- **Best use case:** the single most teachable idea. **Best for:** saves.

## Format 3 — Animated PDF page / page-flip
- **Visual structure:** a printable asset (Permission Slip, "Which AI" sheet, cut-out cards, color-coded age stages) flipping / zooming with gentle motion.
- **Duration:** 15–25s. **Difficulty:** low–med.
- **Voiceover:** short VO reading the key lines.
- **Captions:** minimal — let the page be the star.
- **B-roll:** export PDF pages to images; animate with simple pan/zoom.
- **Best use case:** "save this" reference moments. **Best for:** saves / shares.

## Format 4 — Text-led relatable story
- **Visual structure:** sequence of cream text cards, one line each, calm music or soft VO. The emotional pillar.
- **Duration:** 20–35s. **Difficulty:** low.
- **Voiceover:** optional; if used, warm and unhurried.
- **Captions:** the on-screen text *is* the content — big serif, one thought per card.
- **B-roll:** none needed.
- **Best use case:** "you're not behind" / fears. **Best for:** trust / comments.

## Format 5 — Everyday scenario + AI-generated B-roll
- **Visual structure:** set the scene (5 p.m. kitchen, the mailbox, a waiting room) → the task → the prompt → the payoff. **Objects and hands only — no identifiable people.**
- **Duration:** 25–40s. **Difficulty:** med.
- **Voiceover:** first-person, gentle ("It's 5 o'clock and I've got nothing planned…").
- **Captions:** support the VO; show the prompt as a monospace card.
- **B-roll:** AI-generated or royalty-free object/scene footage; screen capture for the prompt.
- **Best use case:** relatability + a real win. **Best for:** reach / relatability.

## Format 6 — "Which AI should I use?" comparison
- **Visual structure:** a task appears → route it to a "first pick" tool, with a "backup" → repeat for 2–3 tasks. Use the fridge-sheet layout.
- **Duration:** 25–35s. **Difficulty:** low–med.
- **Voiceover:** "If you're trying to ___, start with ___."
- **Captions:** task → tool chips.
- **B-roll:** the "Which AI" sheet or the site's tool picker.
- **Best use case:** tool overwhelm. **Best for:** saves / clicks.
- ⚠️ **Keep it evergreen:** route by *what a tool is good at* in plain terms; avoid quoting prices/models (verify that week if you must).

## Format 7 — Checklist / 3-step tutorial
- **Visual structure:** a numbered list builds on screen (4 habits, PAUSE, "never paste these 5") — one item per beat.
- **Duration:** 20–30s. **Difficulty:** low.
- **Voiceover:** crisp, one line per item.
- **Captions:** big numerals in terracotta; monospace labels.
- **B-roll:** none needed.
- **Best use case:** memorable frameworks. **Best for:** saves.

---

## Turning any format into a repeatable template
1. Build **three reusable canvases** (title card, content card, end card) in your editor once. Every video reuses them.
2. Keep a **prompt-card component** (dark rounded box, monospace, terracotta brackets) — reuse across Formats 1, 2, 5, 6.
3. Keep a fixed **end card**: logo dot + "AI, Made Friendly" + the video's CTA + "link in bio."
4. Batch by format, not by day — record five Format-2 before/afters in one sitting, five Format-1 captures in another.
